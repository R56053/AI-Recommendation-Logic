# AI Recommendation System

## Project Overview

The AI Recommendation System is a Python-based application that recommends items based on user interests. It uses keyword matching and similarity logic to provide personalized recommendations.

## Features

- User Registration
- User Login
- Guest Mode
- AI Recommendation Engine
- Recommendation History
- Favorites
- Ratings
- Admin Panel
- Recommendation Statistics

## Technologies Used

- Python 3
- JSON
- File Handling
- Modular Programming
